# Contributing to tosijs-platform

PRs and issues are welcome!

## Issues

Found a bug or have a feature request? [Open an issue](https://github.com/tonioloewald/tosijs-platform/issues).

## Pull Requests

1. Fork the repo
2. Create a branch (`git checkout -b my-fix`)
3. Make your changes
4. Run `bun format` to format code
5. Run tests: `cd functions && bun test`
6. Submit a PR

## Development Setup

```bash
git clone https://github.com/tonioloewald/tosijs-platform.git
cd tosijs-platform
bun install
cd functions && bun install
```
